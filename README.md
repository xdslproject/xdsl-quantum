# xdsl-quantum

This project provides abstractions and transformations that may be useful when implementing a compiler for quantum programs using [xDSL](https://xdsl.dev/), a Python-native, MLIR-compatible compiler framework.

Our objective is to enable a quantum compiler ecosystem, by sharing the load of implementing common abstractions, and translations between existing formats for quantum programs and compiler intermediate representations.

To contribute to the project and participate in the discussion, please join us on the `Quantum` channel on the [xDSL Zulip](https://xdsl.zulipchat.com/).
